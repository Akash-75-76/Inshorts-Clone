# Inshorts Clone (MERN + MUI)

A news aggregation web application inspired by Inshorts, built using the MERN stack with a clean and responsive UI using Material UI.

---

## Tech Stack
- React
- Material UI (MUI)
- Node.js
- Express.js
- MongoDB
- Mongoose

---

## Features
- Display latest news in short-card format
- Responsive UI with Material UI
- Backend API for news data
- MongoDB for data storage

---

### Clone the repo
